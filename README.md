# Jesvin Sebastian Portfolio

This repository now includes a simple static portfolio site built with plain HTML and CSS.

## Files

- `index.html` - main portfolio page
- `styles.css` - site styling
- `assets/jesvin.jpg` - profile photo
- `Resume_Jesvin_Sebastian.pdf` - downloadable resume

## Deploy On GitHub Pages

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and the `/ (root)` folder.
6. Save the settings.

GitHub Pages will publish the site using `index.html` from the repository root.
